# AI Quiz App

A full-stack quiz app using:
- React Native + Expo
- Express.js backend
- JWT Authentication
- MongoDB
- AI-powered quiz generation

Features:
✔ Login & Signup  
✔ Generate quizzes  
✔ Save quiz history  
✔ Profile & logout  
✔ Percentage & emoji feedback  

To run:

cd backend
npm install
npm start

cd frontend
npm install
npm start
